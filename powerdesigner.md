##文字修改（大小，颜色）

Tools -----> Display Preference.... ------> Format ------> Table -----> Modify ------> Font
然后对所有值做修改（建议字体大小都是12，颜色蓝色最好，主键为红色，外键绿色）；
确定后，点击“Set As Default”！（一定做这个操作哦！）
